# Design: Celestial Tarot Site

## Brand Identity
- **Name:** Celestial Tarot
- **Tone:** Mystical, elegant, cosmic
- **Primary BG:** #0a0a1a (deep navy/black)
- **Accent:** #d8ecf0 (celestial ice blue)
- **Secondary:** #1a1a3e (dark purple-navy)
- **Text:** #e8e8f0 (soft white)
- **Gold accent:** #c9a84c (for stars/highlights)

## Site Structure

```
/                  → Homepage
/zodiac/aries      → Aries page
/zodiac/taurus     → Taurus page
... (12 signs)
/about             → About page
/contact           → Contact page
```

## Page Designs

### Homepage
1. **Hero Banner** — Full-width starfield background, animated subtle twinkle, brand name in serif font, tagline "Unlock the Mysteries of the Stars"
2. **Latest Video** — Embedded YouTube player, featured reading
3. **Zodiac Grid** — 12 zodiac signs as clickable cards with symbols
4. **Brand Intro** — Short paragraph about Celestial Tarot
5. **Social Links** — YouTube, Twitter (@Celestia1Tarot), TikTok
6. **Newsletter** — Email signup form "Get Your Weekly Cosmic Reading"
7. **Footer** — Links, copyright, social icons

### Zodiac Sign Pages (/zodiac/[sign])
- Hero with zodiac symbol and constellation art
- Sign dates and element info
- Embedded YouTube video (sign-specific reading)
- Written tarot reading / horoscope text
- Link to next/previous sign
- CTA to subscribe

### About Page
- Brand story
- Mission statement
- Team/creator info placeholder

### Contact Page
- Contact form (name, email, message)
- Social media links
- Email address

## Typography
- Headings: Playfair Display (serif, elegant)
- Body: Inter (clean, readable)

## Responsive
- Mobile-first design
- Breakpoints: 640px, 768px, 1024px, 1280px
